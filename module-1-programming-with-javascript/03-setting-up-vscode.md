# Setting Up Visual Studio Code for JavaScript

## Overview

Before you can start programming with JavaScript, you need to set up the right development tools. This setup includes installing **Visual Studio Code (VS Code)**, **Node.js**, and the **Code Runner** extension on a Windows operating system.

By completing this setup, you will be able to:

* Write JavaScript code
* Run JavaScript files directly in VS Code
* Understand basic output using `console.log`

---

## What Is Visual Studio Code?

* A lightweight and powerful **code editor**
* Widely used by developers for JavaScript and other languages
* Supports extensions to enhance functionality

---

## Installing Visual Studio Code

### Steps

1. Open a browser and search for **Visual Studio Code**
2. Go to the official website:
   [https://code.visualstudio.com](https://code.visualstudio.com)
3. Click **Download for Windows**
4. Once downloaded, open the installer
5. Accept the license agreement
6. Use default installation settings
7. Select additional options:

   * Create a desktop icon
   * Add "Open with Code" to file context menu
   * Add "Open with Code" to directory context menu
8. Click **Install**
9. Select **Launch Visual Studio Code** and click **Finish**

---

## Installing the Code Runner Extension

### Steps

1. Open VS Code
2. Click the **Extensions** icon on the left sidebar
3. Search for **Code Runner**
4. Install the extension
5. Confirm installation by checking the message:
   *This extension is enabled globally*

---

## Installing Node.js

### Steps

1. Open a browser and search for **Node.js**
2. Visit the official website:
   [https://nodejs.org](https://nodejs.org)
3. Download the Windows version
4. Open the installer
5. Accept the license agreement
6. Click **Install**
7. Click **Finish** once installation completes

Node.js allows JavaScript to run outside the browser.

---

## Creating and Running a JavaScript File

### Creating a File

1. Open VS Code
2. Double-click in the file area to create a new file
3. Click **Select Language**
4. Choose **JavaScript**

---

## Writing Your First JavaScript Code

### Example

* Type the following in the editor:

  * `console.log('Hello World')`

### Running the Code

* Click the **Play** button
* Or use the shortcut: `Ctrl + Alt + N`

### Output

* The message **Hello World** appears in the Output panel

---

## What Does `console.log` Do?

* Displays messages in the console
* Used for debugging and testing code
* Helps developers understand program behavior

---

## Key Takeaways

* VS Code is a powerful editor for JavaScript development
* Node.js enables JavaScript execution outside the browser
* Code Runner allows quick execution of JavaScript files
* `console.log` is used to print output

You are now fully set up to start programming with JavaScript in Visual Studio Code.
